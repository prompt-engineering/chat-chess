## 本地搭建

1. 从 GitHub 克隆 [ChatChess](https://github.com/prompt-engineering/chat-chess)。
2. 执行 `npm install`
3. 运行 `npm run dev` 开启前端
4. 新建控制台，进入 `server` 文件夹
5. 执行 `npm install`
6. 运行 `npm run start` 开启后端
