## Local Usage

1. Clone [ChatChess](https://github.com/prompt-engineering/chat-chess)。
2. Run `npm install`
3. Run `npm run dev` to serve front-end
4. In a new console, cd into `server` directory
5. Run `npm install`
6. Run `npm run start` to start the backend server
